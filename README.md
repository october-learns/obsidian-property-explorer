# Property Explorer

This is an [Obsidian](https://obsidian.md/) plugin that provide an expandable right-pane view that lets you explore all properties across your vault.

## Demo 
![demo](/media/demo.gif)


## How to use 

You can activate this plugin within Obsidian by doing the following:
- Open Obsidian 
- Open Setting > Community plugins
- Turn on community plugins if you haven't 
- Browse > Search for **Property Explorer** > Install > Enable


## Latest Release 

### 0.1.0 
- Features
	- 3-level tree: Property key → unique values → notes with that value
	- Click to open any note in the list
	- Search across keys, values, and note titles simultaneously, with scope filter
	- Sort by name (A→Z / Z→A) or frequency (high→low / low→high)
	- Live updates — re-indexes automatically when notes are created, edited, renamed, or deleted
